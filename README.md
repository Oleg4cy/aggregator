# Aggregator

Сайт-агрегатор услуг для ломбардов.

Проект разрабатывался как каталог организаций с возможностью поиска, фильтрации и просмотра карточек ломбардов по городам, категориям и услугам. Основная задача — собрать данные о компаниях, структурировать их и предоставить пользователю удобный интерфейс для выбора подходящей организации.

## Что реализовано

- Каталог организаций с привязкой к регионам, городам, районам и станциям метро.
- Карточка организации с контактами, графиком работы, услугами и ценами.
- Фильтрация ломбардов по локации, категориям и связанным параметрам.
- Страницы списка организаций и детальной карточки.
- Импорт данных из Excel-файлов.
- Обработка связанных данных: телефоны, сайты, соцсети, почта, график работы, отзывы.
- Административная часть на базе Orchid Platform.
- Frontend-сборка через Vite.

## Особенности проекта

Проект включает не только публичную часть каталога, но и внутреннюю структуру для хранения и обработки данных: организации, услуги, категории, подкатегории, города, районы, метро, цены и графики работы.

Отдельное внимание уделено импорту и нормализации данных из Excel-таблиц, так как исходная информация по организациям могла приходить в разном виде и требовала приведения к единой структуре.
